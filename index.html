What is a PEP?

A PEP, or Portable Encapsulated Project, is a project that subscribes to the PEP structure for organizing project and sample metadata. This structure employs a simple yaml + tsv format that can then read by PEP-compatible tools, starting from our universal R and Python packages. Stop fretting over non-standard annotations. Switch to PEP and immediately inherit complete metadata control in the data analysis environment of your choice.

Data analysts

Start by describing your project samples in PEP format, and immediately benefit from metadata organization tools written in either R or Python.

Software developers

Outsource sample metadata formatting and parsing by using our R or Python APIs, making your tool immediately applicable to any PEP-compliant project.
